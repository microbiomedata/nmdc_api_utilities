External Notebook Examples
==========================

This page highlights external notebook exemplars from the `nmdc_notebooks <https://github.com/microbiomedata/nmdc_notebooks>`_ repository.
Use them as end-to-end learning paths for discovery, integration, and analysis workflows using this package.

If you find any issues with the notebooks, please consider contributing a fix or opening an issue in the `nmdc_notebooks <https://github.com/microbiomedata/nmdc_notebooks>`_ repository.
